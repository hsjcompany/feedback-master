# Framework Selection — 상황별 프레임워크 선택 가이드

> SKILL.md의 STAGE 3에서 이 파일을 참조하세요.
>
> 5개 프레임워크(I-Message, NVC, DEAR MAN, SBI-I, Radical Candor)는
> **배타적 대안이 아니라 연속체**입니다. 대부분의 실전 대화는
> 1개 주 프레임워크 + 1개 보조 프레임워크의 조합으로 만들어집니다.

## 의사결정 4축

인터뷰에서 수집한 정보를 아래 네 축으로 매핑하세요.

1. **정서 온도** — 나·상대의 감정이 얼마나 격한가? (낮음 / 중간 / 높음)
2. **관계 자본** — 쌓아온 신뢰가 얼마나 두터운가? (얕음 / 중간 / 깊음)
3. **목표 우선순위** — 무엇이 가장 중요한가? (a) 결과 / (b) 관계 / (c) 자존감
4. **시간·준비 자원** — 즉석 발화인가, 준비 가능한가? (즉석 / 준비 가능)

## 결정 트리 (Decision Tree)

```
정서 온도 = 높음 (위기 수준)
  → STOP. 진정 기술(TIPP, STOP, 자기공감) 먼저.
  → 다음 기회에 구조적 대화. 지금은 안전 확보.

정서 온도 = 낮음~중간
  ├─ 시간 = 즉석, 목표 = 짧은 업무 정정
  │   → SBI-I (주) + Radical Candor 자세 (보조)
  │
  ├─ 시간 = 즉석, 목표 = 가벼운 관계 피드백
  │   → I-Message (주)
  │
  ├─ 시간 = 준비 가능, 목표 = 결과 달성 (요청/거절/경계)
  │   ├─ 관계 깊음 → DEAR MAN + GIVE
  │   ├─ 관계 중간 → DEAR MAN (단독)
  │   └─ 관계 얕음 or 자존감 압박 → DEAR MAN + FAST
  │
  ├─ 시간 = 준비 가능, 목표 = 관계 회복
  │   └─ NVC 4요소 (주) + GIVE (보조)
  │
  ├─ 시간 = 준비 가능, 목표 = 반복 갈등 해소
  │   └─ NVC 4요소 (주) + DEAR MAN (부탁 단계 보조)
  │
  └─ 시간 = 준비 가능, 성과 리뷰·1on1
      └─ SBI-I + 질문 (기본) + NVC 심화 (필요시)
```

## 혼합 원리 — "연속체(Continuum)"

숙련된 사용자는 세 기법을 동심원으로 중첩 사용한다.

- **바깥 원 — I-Message/SBI-I (1분 대화)**: 모든 발화의 기본 문법.
- **중간 원 — DEAR MAN (10분 대화)**: 이해관계 있는 준비된 요청의 골격.
- **안쪽 원 — NVC (30분+ 대화)**: 관계의 질을 바꾸는 심화 대화의 틀.
- **자세 틀 — Radical Candor**: 위 어느 원에서든 자세로 유지.

## 선택을 사용자에게 설명하는 법

**프레임워크 이름을 강요하지 말 것**. 아래 형식을 권장.

> "말씀하신 상황을 보면 (1) 구체적인 요청을 명료하게 해야 하고, (2) 상대와 오래 일해야 하는 관계이고, (3) 정서가 조금 흔들릴 수 있을 것 같아요.
>
> 그래서 '단계별로 차분하게 요청하는 방식(DEAR MAN)' 을 기본으로 쓰고, 중간중간 '상대의 사정을 인정하는 말(GIVE)' 을 살짝 섞는 게 좋겠어요.
>
> 이 방향이 맞을까요?"

사용자가 "그래요"라고 하면 STAGE 4(스크립트 생성)로 이동. "다른 방식이 좋겠다"고 하면 짧게 대안(예: NVC 심화)을 제안.

## 사용자가 특정 프레임워크를 요청할 때

사용자가 "NVC로 해 주세요" 같이 프레임워크를 지정하면 그대로 따르세요. 단, 부적합하다고 판단되면 **한 번만** 대안을 제시:

> "NVC도 좋은 선택이에요. 다만 이 상황은 결과 요청이 핵심이라 DEAR MAN이 더 안전할 수 있어요. 둘 다 보여드리고 고르시겠어요?"

사용자가 고집하면 그 프레임워크로 진행하세요. 존중이 우선.

## 피해야 할 조합

- **SBI-I + 심층 욕구 탐색**: 과잉. 짧은 피드백은 짧게.
- **NVC 4요소 + 즉석 복도 대화**: 어색함. 준비된 자리로 미루기.
- **DEAR MAN 단독 + 깊은 감정 갈등**: 스크립트가 냉정하게 들림. NVC 선행 필요.
- **Radical Candor 단독**: 자세만 있고 문장이 없어 실행 불가.

인터뷰에서 수집한 정보 없이 기본값을 제시할 때:

- **처음 어려운 대화하는 일반인** → DEAR MAN 스크립트 작성 경험
- **가족·연인 관계** → NVC + GIVE
